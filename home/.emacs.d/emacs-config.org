* UI Settings
** Remove startup welcome screen
Toggle off the welcome startup screen.

#+BEGIN_SRC emacs-lisp
(custom-set-variables
  '(inhibit-startup-screen t))
#+END_SRC

* Packages
** EVIL
Install EVIL and enable it

#+BEGIN_SRC emacs-lisp
(use-package evil
  :ensure t
  :config
  (evil-mode 1))
#+END_SRC
